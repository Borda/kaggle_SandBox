# Resources

This folder is intended for lightweight notes, links, metadata, and references required for the project.

Do not commit Kaggle datasets, generated submissions, checkpoints, or pretrained model weights here. Keep those files in ignored local folders such as `data/`, `submissions/`, `checkpoints/`, or `models/`, and document how to reproduce or download them.
